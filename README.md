# 🚀 AI Website Copy Generator for Local Businesses

🔗 **Live Website:** [https://careplus-healthease-hub.lovable.app]

---

## 📌 Project Overview

Most local businesses lose customers not because of poor service, but because their website fails to clearly communicate value.

This project solves that problem using **Prompt Engineering + AI tools** to generate **high-converting website copy** and transform it into a real, visually appealing website.

---

## 🏥 Business Case Study

**CarePlus Diagnostic Center**
📍 Jalandhar, Punjab, India

🎯 Target Audience:

* Families
* Working professionals
* Elderly patients

---

## 🎯 What This Project Does

This system generates:

✅ Clear & compelling homepage copy
✅ Service descriptions with benefits
✅ High-converting call-to-action (CTA) sections

All content is:

* Conversion-focused
* Business-specific (not generic AI text)
* Ready to publish on real websites

---

## 🧠 Prompt Engineering Approach

A reusable **master prompt system** was designed to take structured inputs:

* Business name
* Type
* Location
* Services
* Target audience

👉 And generate complete website-ready content.

---

## 🛠️ Tools Used

* **ChatGPT** → Prompt Engineering & Content Generation
* **Framer AI** → Website UI/UX Design
* **Lovable.dev** → AI Website Builder

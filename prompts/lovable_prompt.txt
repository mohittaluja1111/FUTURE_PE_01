PROMPT 2 :- FIX PROMPT (FOR BETTER TEXT VISIBILITY)

Improve the HERO SECTION design to make text highly readable and visually strong.

* Add a dark gradient overlay on the background (blue → dark blue or black at 40–60% opacity)
* Ensure the background is slightly darker so white text stands out clearly

### TEXT IMPROVEMENTS:

* Headline:

  * Use pure white color (#FFFFFF)
  * Increase font weight (bold / extra bold)
  * Add slight text shadow or glow for better readability

* Subheadline:

  * Use off-white color (#E5E7EB)
  * Increase font size slightly
  * Improve line spacing

### BUTTON IMPROVEMENTS:

* Primary button (Book Now):

  * Use solid high-contrast color (dark blue or gradient)

* Secondary button (Call Now):

  * Add border with strong contrast (white or dark blue)
  * Increase thickness of border

### TRUST BADGES:

* Make badges more visible:

  * Add solid or semi-dark background
  * Increase icon and text contrast

### OVERALL:

* Ensure accessibility (high contrast between text and background)
* Keep design premium but prioritize readability
* Maintain modern UI with clear visual hierarchy
